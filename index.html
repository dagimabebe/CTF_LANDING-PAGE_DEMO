Challenge Categories

Test your skills across multiple security domains. Each category presents unique challenges that will push your technical abilities to the limit.

Web Exploitation

Exploit vulnerabilities in web applications including XSS, SQLi, CSRF, and more. Can you bypass security controls and gain unauthorized access?

$ curl -X POST vulnerable.site/api/admin

Cryptography

Break encryption schemes, analyze ciphertexts, and solve complex crypto puzzles. Can you crack the code and uncover hidden messages?

$ openssl aes-256-cbc -d -in flag.enc

Binary Exploitation

Find vulnerabilities in executable binaries and exploit them to gain control. Buffer overflows, ROP chains, and shellcode await.

(gdb) run $(python -c 'print "A"*256')

Reverse Engineering

Analyze compiled programs without source code. Understand their behavior and extract hidden information.

$ r2 -AAA ./challenge

Forensics

Investigate digital artifacts, recover deleted files, analyze network traffic, and uncover hidden data.

$ binwalk -e suspicious.img

Miscellaneous

Unique challenges that don't fit other categories. OSINT, steganography, and other creative puzzles.

$ steghide extract -sf image.jpg
